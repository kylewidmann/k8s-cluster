Helm Chart Repository

This is a Helm chart repository hosted on GitHub Pages for Kubernetes deployment manifests.

Usage

To use this repository with Helm:

helm repo add k8s-cluster https://kylewidmann.github.io/k8s-cluster/repo
helm repo update

Then you can see the available charts:

helm search repo k8s-cluster

And install a chart:

helm install my-release k8s-cluster/CHART_NAME

Available Charts

monitoring

Monitoring stack for Kubernetes using Prometheus, Grafana, and other tools.

helm install monitoring k8s-cluster/monitoring

Repository

The index file is available at index.yaml.
